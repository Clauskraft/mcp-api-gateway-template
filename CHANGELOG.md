# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-18

### Added
- Initial release of MCP API Gateway Template
- TypeScript-based MCP server implementation
- Modular API configuration system
- HTTP request utilities with query and path parameter support
- Pre-configured OpenWeatherMap API integration
- Pre-configured JSONPlaceholder API for testing
- Comprehensive documentation (README, Quick Start, Contributing, Template Usage)
- Example configuration files for Claude Desktop
- MIT License
- Environment variable support for API keys
- Build and development scripts
- .gitignore for Node.js projects

### Features
- Support for multiple API endpoints
- Automatic tool generation from API configurations
- Type-safe parameter handling
- Error handling and reporting
- Environment-based API key injection
- Clean separation of concerns (APIs, types, utilities)

### Documentation
- Comprehensive README with setup instructions
- Quick Start guide for rapid onboarding
- Template usage guide for customization
- Contributing guidelines
- Example configurations
